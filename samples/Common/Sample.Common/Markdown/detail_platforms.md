## All Platforms

MvvmNav is designed to work across all major platforms supported by .NET.

### Supported platforms

- 🪟 Windows  
- 🐧 Linux  
- 🍎 macOS  
- 📱 iOS  
- 🤖 Android  

### Supported UI frameworks

MvvmNav is UI-agnostic and works with the most important .NET UI technologies:

- 🧊 **WPF** (Windows)
- 🌐 **Avalonia** (cross-platform)
- 📱 **.NET MAUI** (mobile & desktop)
- 🧩 **Uno Platform** (WinUI, WebAssembly, and more)

### Key idea

👉 The navigation logic lives in a **platform-independent core**.

👉 UI-specific behavior is handled by lightweight platform integrations.

### Benefits

- Write navigation logic once, reuse everywhere  
- Keep ViewModels independent from UI frameworks  
- Use the same patterns across desktop and mobile  

MvvmNav adapts to your platform — not the other way around.
